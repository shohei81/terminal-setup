Set up your terminal
in minutes.

A single command gives you a modern, beautiful terminal on macOS — no config files, no guesswork.

curl -fsSL https://raw.githubusercontent.com/shohei81/terminal-setup/main/install.sh | sh
macOS only · Requires an internet connection

What you get

✦

Beautiful by default

Tokyo Night theme, Hack Nerd Font, and Starship prompt — configured automatically. Looks great from the first launch.

⚡

Modern tools included

lsd, bat, fd, ripgrep and more — the latest CLI tools that make everyday terminal work faster and clearer.

◎

Your choices, zero config

Pick your theme, font size, and optional tools through a simple interactive menu. No config files to write.

How it works

Run the command

Paste one line into your terminal. Homebrew is installed automatically if needed.

Choose your style

Pick a color theme, font size, and whether the background should be transparent.

Pick your tools

Select optional tools like fzf, zoxide, yazi, and more from an interactive list.

Done

Restart Ghostty. Your new terminal environment is ready.
