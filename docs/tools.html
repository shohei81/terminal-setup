Tools

Everything terminal-setup installs, and why.

Always installed

These tools are installed automatically for everyone. They form the foundation of a modern, comfortable terminal experience.

Ghostty

A fast, feature-rich terminal emulator built for macOS. GPU-accelerated with native look and feel.

github.com/ghostty-org/ghostty →

Hack Nerd Font

A monospace font patched with thousands of icons — used by Starship, lsd, and many terminal tools.

github.com/ryanoasis/nerd-fonts →

Starship

A minimal, blazing-fast shell prompt that shows Git status, language versions, and more — automatically.

github.com/starship/starship →

lsd

A modern replacement for ls. Shows icons, colors, and better formatting out of the box.

github.com/lsd-rs/lsd →

bat

A replacement for cat with syntax highlighting, line numbers, and Git change indicators.

github.com/sharkdp/bat →

fd

A faster, friendlier replacement for find. Easier to remember and 5–50× faster.

github.com/sharkdp/fd →

ripgrep

Extremely fast text search through files. Used internally by many editors and AI tools.

github.com/BurntSushi/ripgrep →

fastfetch

Displays a clean system overview (OS, CPU, RAM, etc.) each time you open a new terminal.

github.com/fastfetch-cli/fastfetch →

zsh-autosuggestions

Shows ghost-text command suggestions as you type, based on your history. Tab to accept.

github.com/zsh-users/zsh-autosuggestions →

zsh-completions

Adds hundreds of extra Tab-completion definitions for popular tools.

github.com/zsh-users/zsh-completions →
Optional — your choice

These tools are powerful but have a learning curve. Install the ones that match how you work.

fzf

A fuzzy finder for the terminal. Press Ctrl+R to search command history interactively, or pipe any list through it.

github.com/junegunn/fzf →

zoxide

Replaces cd with a smarter version that learns your most-visited folders. Type z project instead of a full path.

github.com/ajeetdsouza/zoxide →

yazi

A visual file manager that runs entirely in the terminal. Browse, preview, and open files without leaving your shell.

github.com/sxyazi/yazi →

lazygit

A visual Git interface in the terminal. Stage changes, write commits, and manage branches — all without memorizing Git commands.

github.com/jesseduffield/lazygit →

bottom

A beautiful, interactive system monitor. View CPU, memory, and process usage in real time. Run with btm.

github.com/ClementTsang/bottom →
