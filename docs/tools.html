Always installed

These tools are installed automatically for everyone.

Ghostty

A fast, feature-rich terminal emulator built for macOS. GPU-accelerated with a native look and feel. terminal-setup configures it automatically with your chosen theme and font.

Open it like any Mac app — from Launchpad, Spotlight (Cmd+Space → type "Ghostty"), or your Dock.

github.com/ghostty-org/ghostty →

Hack Nerd Font

A monospace font patched with thousands of icons. Used by Starship, lsd, yazi, and many other tools to show file icons, Git status symbols, and branch indicators in the terminal. Applied automatically in Ghostty — no commands needed.

github.com/ryanoasis/nerd-fonts →

Starship

A minimal, fast shell prompt that shows useful context automatically — the current Git branch, whether there are uncommitted changes, which language version a project uses, and more. Loads silently every time you open a terminal.

# Shown automatically in your prompt — no command to run
~/projects/myapp git:(main ✗) ▸
github.com/starship/starship →

lsd

A modern replacement for the ls command, which lists files in a directory. lsd adds color-coded output, file icons from Nerd Font, and better formatting — making it much easier to scan a folder at a glance.

$ lsd # list files with icons
$ lsd -la # show hidden files + details
$ lsd --tree # show folder as a tree
github.com/lsd-rs/lsd →

bat

A replacement for the cat command, which prints file contents to the terminal. bat adds syntax highlighting (code is colored by language), line numbers, and Git change indicators — so you can read files without opening an editor.

$ bat README.md # view a file with highlighting
$ bat main.go # Go code with syntax colors
github.com/sharkdp/bat →

fd

A faster, friendlier replacement for the find command. Searches for files and folders by name — much simpler to type than find, and typically 5–50× faster.

$ fd config # find files named "config"
$ fd .py # find all Python files
$ fd .log ~/Library # search in a specific folder
github.com/sharkdp/fd →

ripgrep

Searches the text inside files, extremely fast. Run as rg. Used internally by many code editors and AI tools — if Claude or Cursor searches your codebase, it often uses ripgrep under the hood.

$ rg "TODO" # find "TODO" in all files here
$ rg "api_key" ~/ # search across your home folder
github.com/BurntSushi/ripgrep →

fastfetch

Displays a system info summary every time you open a new terminal — macOS version, CPU, memory, uptime, and more. You choose which items to show during setup.

$ fastfetch # show system info manually
github.com/fastfetch-cli/fastfetch →

zsh-autosuggestions

As you type, shows a faint ghost-text suggestion based on your command history. Press → (right arrow) or Tab to accept it. Over time it learns your most-used commands and suggests them immediately.

# Type the start of a command you've run before:
$ git push origin main ← press → to accept
github.com/zsh-users/zsh-autosuggestions →

zsh-completions

Adds hundreds of extra Tab-completion definitions for popular tools. Press Tab while typing a command to see available options, subcommands, or file paths. Works automatically — no commands needed.

# Press Tab to complete commands and options:
$ git Tab # shows: add, commit, push, pull …
github.com/zsh-users/zsh-completions →
Optional — your choice

Powerful tools with a learning curve. Install what sounds useful — you can add more later.

fzf

A fuzzy finder — type a few letters and it narrows down a list interactively. Most useful for command history search: press Ctrl+R to search everything you've ever typed, then select and re-run it.

$ fzf # interactively filter any list
Ctrl+R # fuzzy-search command history
$ ls | fzf # filter file list and select one
github.com/junegunn/fzf →

zoxide

A smarter replacement for the cd command. Learns which folders you visit most often, so you can jump to them by typing just a few letters — no full path needed.

$ z proj # jump to ~/projects/myapp
$ z down # jump to ~/Downloads
$ zi # interactive folder picker
github.com/ajeetdsouza/zoxide →

yazi

A visual file manager that runs inside the terminal. Browse folders, preview files, and open or move things — all with keyboard navigation, without ever leaving the terminal.

$ yazi # open file manager here
# h/j/k/l to navigate · Enter to open · q to quit
github.com/sxyazi/yazi →

lazygit

A visual Git interface in the terminal. Stage files, write commit messages, view diffs, manage branches — all without memorizing Git commands. A good starting point for learning Git without the command-line friction.

$ lazygit # open inside a Git repo
# space to stage · c to commit · P to push · q to quit
github.com/jesseduffield/lazygit →

bottom

An interactive system monitor for the terminal. Shows CPU usage, memory, disk, network, and running processes in real time — like Activity Monitor, but in the terminal and keyboard-driven.

$ btm # open system monitor
# q to quit · arrow keys to navigate
github.com/ClementTsang/bottom →
