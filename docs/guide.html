Setup Guide

A walkthrough of what happens when you run terminal-setup.

1. Install

Paste this into any terminal window (Terminal.app works fine):

curl -fsSL https://raw.githubusercontent.com/shohei81/terminal-setup/main/install.sh | sh

The script will:

  • Check that you're on macOS
  • Install Homebrew if it's not already installed
  • Download the terminal-setup binary for your Mac (Apple Silicon or Intel)
  • Launch the interactive setup

2. Choose your appearance

The first set of screens lets you customize how Ghostty looks.

Color theme

Pick from 10 popular themes shown with a live preview of each palette. Themes range from dark to light, cool to warm.

Font size

Choose a font size that feels comfortable. The default is 14pt. Ghostty uses Hack Nerd Font, which is installed automatically.

Background opacity

Enable background transparency if you want a frosted-glass look. Works best with a colorful desktop wallpaper.

3. Terminal info display

fastfetch shows a system overview when a new terminal opens. Choose which details to show:

  • OS — macOS version
  • Host — your Mac model
  • Shell — your current shell (zsh)
  • CPU — processor name and speed
  • Memory — RAM usage
  • Battery — charge level and estimated time
  • Disk — available storage
  • GPU — graphics chip
  • Uptime — how long since last restart
  • Colors — a color palette swatch

Navigate with ↑↓, toggle with Enter, then move down to Confirm.

4. Optional tools

A set of powerful extras you can opt into. Each one is independent — install only what sounds useful to you.

See the Tools page for descriptions of each option.

5. .zshrc strategy

terminal-setup needs to add a few lines to your .zshrc file (your shell config). You can choose how:

A

Append (recommended)

Adds terminal-setup's config block to the end of your existing file. A date-stamped backup is saved first.

B

Overwrite

Replaces your entire .zshrc with the generated config. A backup is saved before anything is changed.

C

Skip

Don't touch .zshrc at all. Ghostty and tools will still be installed.

If you run terminal-setup again, the append strategy will find its own config block and update it in place — your other .zshrc content stays untouched.

6. Review & confirm

A summary of your choices is shown before anything is installed. If something looks wrong, press esc to go back.

Confirm to begin installation. Each step prints its progress as it runs.

7. Restart Ghostty

Once installation completes, quit and relaunch Ghostty. Your new theme, prompt, and tools are ready.

If Ghostty isn't installed yet, download it from ghostty.org and open it for the first time.

Running again

terminal-setup is idempotent — safe to run multiple times. Already-installed packages are skipped. Your previous choices are shown as defaults so you only change what you want.
