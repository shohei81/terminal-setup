Setup Guide

A complete walkthrough, from zero to a modern terminal environment.

Before you start

This page explains everything from scratch. No prior experience needed.

What is a terminal?

The terminal is a text-based window where you control your Mac by typing commands instead of clicking. It looks minimal, but it's one of the most powerful tools on your computer. terminal-setup makes the whole setup process interactive — you just answer questions and it does the work.

What is Ghostty?

Ghostty is the terminal application you'll use after setup. It replaces the default Terminal.app that comes with macOS — it's faster, looks better, and supports all the tools terminal-setup installs. terminal-setup installs Ghostty for you automatically.

What is Homebrew?

Homebrew is a free tool that installs developer software on your Mac — similar to an App Store, but for command-line tools. terminal-setup uses Homebrew to install everything. If you don't have it, the installer adds it for you automatically. You don't need to do anything in advance.

1. Open a terminal window

You need any terminal window to run the install command. The default Terminal.app that ships with macOS works fine for this step.

1

Open Spotlight

Press Cmd + Space on your keyboard. A search bar appears in the middle of your screen.

2

Search for Terminal

Type Terminal and press Enter. A window with a plain text prompt opens.

3

Paste and run the install command

Copy the command below, paste it into the Terminal window with Cmd + V, then press Enter.

curl -fsSL https://raw.githubusercontent.com/shohei81/terminal-setup/main/install.sh | sh

The script checks your Mac, installs Homebrew if needed, downloads the terminal-setup program, and launches it. You may be asked to enter your Mac login password — this is normal and needed for Homebrew to install software.

2. Choose your appearance

The first set of screens lets you customize how Ghostty looks. Use the arrow keys to navigate and Enter to select.

Color theme

Pick from 10 popular themes. Each option shows a small preview of the actual colors — you can see the difference before choosing. Themes are split into dark (easier on the eyes at night) and light (better in bright rooms).

Font size

Choose how large the text appears. The default is 14pt — a good starting point. You can always change it later by running terminal-setup again.

Background opacity

If you turn this on, the terminal background becomes slightly see-through, like frosted glass. It's a visual effect — pick whichever you prefer.

3. Terminal info display

fastfetch shows a summary of your Mac's info every time you open a new terminal window. You choose which details to show.

Navigate with ↑↓, toggle items on/off with Enter, then move down to the Confirm button and press Enter to save.

  • OS — macOS version
  • Host — your Mac model name
  • Shell — the shell you're using (zsh)
  • CPU — processor name and speed
  • Memory — RAM usage
  • Battery — charge level
  • Disk — available storage
  • GPU — graphics chip
  • Uptime — time since last restart
  • Colors — a color palette swatch

4. Optional tools

A set of extra tools you can install or skip. Each one is independent — install only what sounds useful. You can add more later by running terminal-setup again.

See the Tools page for a description of each option.

5. Shell config (.zshrc)

terminal-setup needs to add a small block of settings to a file called .zshrc. This file runs automatically every time you open a terminal — it's how tools like Starship and autosuggestions get loaded.

Don't worry if you've never heard of it. Just pick one of the options below:

A

Append — recommended for most people

Adds terminal-setup's settings to the end of your existing file. Your current settings are kept. A backup is saved automatically before anything changes.

B

Overwrite — fresh start

Replaces your entire .zshrc with a new one. Choose this only if you want to start completely from scratch. A backup is saved first.

C

Skip — handle it yourself

terminal-setup won't touch your .zshrc at all. Ghostty and tools will still be installed, but you'll need to add the config lines manually.

If you run terminal-setup again later, the Append option will find its own block and update it in place — your other settings stay untouched.

6. Review & confirm

Before anything is installed, you'll see a summary of all your choices. Read it over — if something looks wrong, press Esc to go back and change it.

When you're ready, confirm to begin. Each step prints its progress as it runs. Installation usually takes 2–5 minutes depending on your internet speed.

7. Open Ghostty

Ghostty is installed automatically during setup. Once the installer finishes:

1

Open Spotlight

Press Cmd + Space, type Ghostty, and press Enter.

2

Allow if asked

macOS may show a security prompt the first time. Click Open to continue.

3

Your new terminal is ready

Your color theme, Starship prompt, and tools are all active. If you see the fastfetch system info at the top, everything worked.

Running it again

terminal-setup is safe to run multiple times. Already-installed packages are skipped, and your previous choices are shown as defaults — so you only change what you want.

terminal-setup

Use this to try a different theme, add more tools, or update your config after learning what you actually use.

Something went wrong?

If the installer fails partway through, it's usually safe to run it again — it picks up where it left off. If you're stuck, open an issue on GitHub with the error message and someone can help.
